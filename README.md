# setup (windows)
- install python 3.11
- `py -m venv bveri`
- `.\bveri\Scripts\activate`
- `python -m install pip --upgrade pip`
- `pip install poetry`
- `cd underwater_imagery`
- `poetry install`
- `python tests\cuda_test.py`
